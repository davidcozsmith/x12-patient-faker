# X12 Patient Faker
Uses [faker](https://www.npmjs.com/package/faker) to generate test patient data in HIPAA X12 format.

Example output:
```
NM1*IL*1*EMARD*SUMMER*F***MI*IXPOSKS1HOQ~
N3*683 LESCH CREST~
N4*SYLVESTERSTAD*NJ*79141-0456~
DMG*D8*19320429*F~
DTP*472*D8*20170929~
```

To start development clone then run:
```
npm install
npm run start
```
